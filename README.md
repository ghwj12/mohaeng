# 모행 (모두의 모든 행사)

Cloud Native 기반 통합 이벤트 관리 플랫폼
**누구나 행사를 주최하고, 누구나 행사에 참여할 수 있는 서비스**

---

## 프로젝트 소개

**모행**은 지역 기반 행사 탐색과 AI 맞춤 추천을 제공하는 이벤트 관리 플랫폼입니다.

사용자는 관심 있는 행사를 쉽게 찾고 참여할 수 있으며,
주최자는 행사를 등록하고 리뷰 데이터를 통해 행사에 대한 피드백을 받을 수 있습니다.

---

## 팀 소개

Team **풀풀**은 풀스택 개발자들로 구성된 팀으로, 열정과 기술력이 *풀풀* 풍기는 팀입니다.

* 김현정 (팀장)
* 정민수
* 박지왕
* 전민제

---

## 기획 배경 및 서비스 차별점

기존 행사 플랫폼은 기업 및 대규모 행사 중심으로 구성되어 있어,
소규모·취미·지역 기반 행사는 노출과 참여가 어렵다는 한계가 있습니다.

또한 행사 종료 후 사용자 경험을 공유하고 데이터로 활용할 수 있는
리뷰 및 피드백 시스템이 부족하다고 판단했습니다.

모행은 이러한 문제를 해결하기 위해 다음과 같은 차별화 요소를 제공합니다.

### 지역 중심 탐색
- 지도 기반 UI를 활용하여 내 주변 행사를 직관적으로 탐색
- \"주제보다 지역\"에 초점을 맞춘 행사 발견 경험 제공

### 스마트 알림 시스템
- 관심 행사 등록 시 행사 전날 및 당일 자동 알림
- 사용자의 관심사를 기억하는 맞춤형 알림 기능

### 리뷰 기반 피드백
- 참여자: 행사 경험과 감정 공유
- 주최자: 리뷰 데이터 분석을 통한 행사 품질 개선

---

## 주요 기능

* 회원가입 / 로그인 (일반 회원가입, Google 연동)
* 지도 UI 및 달력 UI 활용 행사 탐색
* 행사 목록 조회 및 검색 / 필터링
* 행사 등록 및 참가 신청
* 관심 행사 등록 및 알림 설정
* 지도 API 기반 행사 위치 표시
* 행사 리뷰 작성 및 조회
* AI 기반 행사 추천 챗봇
* 관리자 기능 (휴면 계정 관리, 신고된 행사 조치, 행사 통계, 운영 통계)
